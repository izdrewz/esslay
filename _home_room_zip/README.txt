# Home Life Admin Room Placeholders

Copy the `docs/assets/rooms/home-life-admin/` folder into the root of the `izdrewz/esslay` repo.

These are original placeholder backgrounds selected for Home Base Life Admin v1:
- bathroom
- kitchen
- bedroom
- wardrobe/laundry
- desk/focus planner
- front door/errands
- creative nook
- recovery library optional

The repo code references these paths:
`docs/assets/rooms/home-life-admin/<filename>.webp`
