# Area 13 — Source Library and Quote Bank Integration

Area 13 is for future source, reading, note, quote bank, and evidence handling.

## Status

Do not fully build this in Version 0.1. Version 0.1 should only keep space for source and quote-bank features later.

## Reference repo

Use `izdrewz/tmazing` as the future reference for source storage and quote-bank ideas. Do not merge it into `esslay` yet.

## Useful future ideas

`tmazing` may be useful later for:

- module-organised source storage
- unit and block fields
- title, author, year, publisher, URL, file note, and page/location fields
- PDF and DOCX text extraction
- source searching
- draft/source matching
- bibliography and reference support
- local export/import

## Version 0.1

Only keep placeholder planning for:

- Source Mine
- Quote Geodes
- evidence loot
- missing evidence as a monster/problem
- source shelf placeholder in the edit room

## Future source fields

- source id
- module code
- unit/block/topic
- source title
- author
- year/date
- publisher/course material/website
- URL/file note/local location
- page, paragraph, section, or timestamp
- pasted or extracted text
- user notes
- assignment/task link
- usefulness rating
- tags/keywords

## Future quote bank fields

- quote or extract text
- source id
- source title
- author/year if available
- page, paragraph, or section location
- linked assignment question
- linked command word
- linked keyword
- linked module area
- paragraph or section it might support
- user note on why it matters
- whether it has been used in the draft
- whether it needs checking later

## Rule

The app should help collect, label, connect, and reuse evidence. The user stays in control of what is used and where it goes.
