Area 6 is not the current active blocker for Study Cave work.

## Locked main room direction

The main Home Base is the central room the player returns to before clicking into:
- Edit Room
- Study Cave
- Wardrobe/Mirror
- Village
- bedroom/stairs
- future areas

Locked base requirements:
- characterless background
- no UI baked into the image
- no labels baked into the image
- 16:9 game background
- cosy medieval/renaissance cottage interior
- warm scholar-adventurer mood
- stairs to bedroom
- study cave / study room doorway
- fireplace
- stained glass window
- open centre space for character as separate layer
- architecture kept stable for future furniture layers
